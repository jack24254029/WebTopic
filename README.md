# WebTopic

## 專案功能
1. 新增、刪除、修改護士資料
2. 新增、刪除、修改站點資料
3. 分配護士至站點之功能
4. 查看站點已加入護士之功能

## 開發環境
- IDE：Intellij IDEA 2017.3.5 (Ultimate Version)
- Broswer：Google Chrome 64.0.3282.186 (64 bit)
- MySQL：5.7.21
- MySQL Manager：Sequel Pro 1.1.2
